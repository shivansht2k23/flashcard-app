# Flashcard Quiz App

A simple command-line flashcard application to help you learn and memorize information through quizzes.

## Features

- **Add Flashcards**: Create custom question-answer pairs
- **Take Quizzes**: Test your knowledge with randomized questions
- **View All Flashcards**: Browse your entire flashcard collection
- **Persistent Storage**: Flashcards are saved automatically and available between sessions

## How to Use

1. Run the application
2. Choose from the menu options:
   - **Option 1**: Add a new flashcard by entering a question and answer
   - **Option 2**: Take a quiz on all your flashcards
   - **Option 3**: View all saved flashcards
   - **Option 4**: Exit the application

## File Structure

- `main.py` - Entry point for the application
- `flashcards.py` - Core flashcard logic and menu system
- `flashcards.json` - Storage file for your flashcards (created automatically)

## Requirements

- Python 3.11 or higher

## Getting Started

Simply run the application and start adding flashcards. Your progress is saved automatically to `flashcards.json`.
